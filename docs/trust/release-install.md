# Release Install

Normal Codex installs should use the repository marketplace, which points to `release/tokengraph`.

The release plugin includes compiled MCP runtime files and does not require users to run `pnpm install`, `pnpm build`, TypeScript, cloud sync, an OpenAI API key, or an embeddings service.

If MCP tools are missing, confirm Codex installed the release plugin rather than the maintainer source plugin.
